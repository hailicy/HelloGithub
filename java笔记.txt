首行是包名
类名与java文件名一致
float型小数后必须加f
char型使用Unicode编码，可以用编码代替字符，char x = 'a'等价于char x = 97，强制类型转换可以在int和char间相互转换
布尔类型关键字boolean，只有true和false两个值，不能与整数类型转换
常量关键字为final，格式为final double PI = 3.1415926D
类中的变量是成员变量，方法中的是局部变量
成员变量定义前加static可使其跨类使用
局部变量与成员变量名一致时，成员变量被屏蔽
三元运算符格式为 int x = a > b?x:y，为真则取x
强制类型转换格式为 (int)45.23
if else可以考虑用三元运算符替代
字符数组转换为字符串变量格式为a['s','t','u'];String x=new String(a)，等价于String s = new String("stu")
	从数组a的第x个元素开始，长度为y，格式为new String(a,x,y)
	也可为String str1,str2;str1=str2="stu"
+可以连接字符串
获取字符串长度可以使用.length()方法，返回值包含字符串中的空格
查找字符串中某字符或字符串第一次出现的地方用.indexOf("stu")方法，搜最后一次出现的地方用.lastIndexOf("stu")方法
返回指定地方的字符可以用.charAt(3)方法
截取字符串，用.substring(3)方法，截取从第三个到结束
.substring()方法也可以重载为两个参数，第一个是开头的位置，第二个是结尾的位置的后一位，左闭右开
忽略前后空格用.trim()方法
替换可用.replace('e','E')方法，用后者替换前者
.startswith('a')和.endswith()方法用于判断开头结尾是否为参数值，返回值为布尔类型
因为字符串的名字是引用，不能直接用==判断字符串是否相等，否则判断的是地址，要用.equals(str)来比较，返回值为布尔类型
呼略大小写可以用equalsIgnoreCase(str)方法
大小写转换可以用.toLowerCase()方法和.toUpperCase()方法
字符串分割用.split(x,3)方法，前面是分割标志，后面是分割成最终的数量
正则表达式可以用.matches(regex)方法，返回值是布尔类型
	[^456]：代表4、5、6之外的任何字符
	[a-r]：代表a-r中的任何一个字母
	后缀{4}表示重复拼配四次
	[a-zA-Z]：可表示任意一个英文字母
	\\w表示任何字符，\\w+表示任何字符并且出现1次或多次
	\\w{2,4}表示任何字符，出现2到4次
声明一维数组有两种方式，int[] arr和int arr[]，然后分配内存空间，格式为arr=new int[5]，每个元素初始值为0，等价于int arr[]={1,56,4,6
二位数组声明可以是int[][] arr也可以是int arr[][]，可以为int arr[][]={{2,3},{8,6}}

用for循环遍历数组的方式是for(int i:arr)
导入Arrays类后可以用.fill(arr,6)来用6填充整个数组
Arrays可以用.sort(arr)方法对数组进行升序排序，如果是String类数组则数字在最前，小写字母最后

一个java文件中只有一个public类，类名与java文件名一致
定义一个对象格式为STUDENT stu1=new STUDENT()
构造方法前加public
static修饰的成员变量和方法，从属于类。
普通变量和方法从属于对象的。
子类不能继承父类的构造方法
子类可以写参数、名称与父类一样的方法来实现自身的行为替代父亲的行为
super可以访问父类，用法与this相同
	如果父类的构造器有参数，子类的构造器就必须在第一行使用super()调用父类的构造器

取整先import Math类，用Math.floor(num)向下取整，四舍五入则Math.round(num)等价于Math.floor(num+0.5)或者加(int)强转
求绝对值用Math.abs(num)
随机数用Math.random()可以随机取0-1的数，(int)100*Math.random()可以取0-100的数
Math.pow(a,b)a的b次方
Math.sqrt(num)可以求num的平方根
